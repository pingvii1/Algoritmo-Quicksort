# Algoritmo de Ordenamiento: Quicksort

Este repositorio contiene la implementación en Java del algoritmo de ordenamiento **Quicksort**, desarrollada como parte de la asignatura **Estructuras de Datos** en la **Universidad Piloto de Colombia**.

## 🚀 Implementación
La solución se basa en la estrategia de particionamiento de **Lomuto**, utilizando el último elemento como pivote, tal como se documenta en la presentación adjunta.

### Características técnicas:
* **Lenguaje:** Java.
* **Interfaz:** Uso de `Comparable` para permitir la ordenación de cualquier tipo de objeto.
* **Lineamientos:** Implementación de métodos estándar:
  * `less()`: Comparación de elementos.
  * `exch()`: Intercambio de posiciones en el arreglo.
  * `show()`: Visualización del estado del arreglo (Paso a paso).
  * `isSorted()`: Verificación final de ordenación.
